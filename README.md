# Student Registration Form

## Description

This project is a simple HTML Student Registration Form designed to collect student information. It demonstrates the use of common HTML form elements such as text fields, radio buttons, checkboxes, dropdown menus, text areas, fieldsets, and tables.

## Features

* Personal Information section

  * Full Name
  * Email Address
  * Phone Number
  * Date of Birth

* Gender Selection

  * Male
  * Female
  * Other

* Country Selection dropdown

* Subject Selection using checkboxes

  * Science
  * Mathematics
  * English
  * Swahili
  * Music

* Programming Languages multi-select list

  * Java
  * Python
  * C
  * C++
  * JavaScript
  * COBOL

* Comments section for additional information

* Student Information Table displaying:

  * Student ID
  * Course
  * Year

* Form buttons:

  * Register (Submit)
  * Clear Form (Reset)

## Technologies Used

* HTML5

## How to Run

1. Save the file as `index.html`.
2. Open the file in any web browser.
3. Fill in the registration details.
4. Click **Register** to submit the form or **Clear Form** to reset the fields.
